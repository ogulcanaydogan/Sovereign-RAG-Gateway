# v0.9.0-rc1 Release Dossier

Release date: 2026-03-07

## Summary

`v0.9.0-rc1` is the release-candidate cut for the hardening-only path to `v1.0.0`.

No HTTP/API contract changes are introduced. The objective is operational stabilization and GA promotion safety.

## Compatibility

- HTTP API: unchanged.
- Contracts under `docs/contracts/v1/`: backward-compatible.

## Known Limitations

- Stabilization metrics derive from GitHub Actions run history; run retention policy affects historical visibility.
- Rollback drill remains a controlled scripted drill and is not yet parameterized for arbitrary version targets.
- Weekly evidence report snapshots are summary artifacts, not full dashboard exports.
